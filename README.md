# Memory-Game
Memory Game for ICS-OS
--------------------------------

1. 	Copy the folder 'memory' to 'ics-os/contrib' directory 	

2. 	Go to 'ics-os/contrib/memory' directory
		$make install 

3. 	Go back to 'ics-os' directory
		$make floppy
		$make boot-floppy

4. 	Run memory.exe inside ics-os
		$memory.exe
